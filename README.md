# Игра чет-нечет

## Настрока окружения
Для работы с проектом необходимо установить [Angular CLI](https://github.com/angular/angular-cli#installation)

## Запуск dev сервера
```bash
npm install
ng serve
```

## Сборка для развертывания
```bash
ng build
```
Содержимое dist разместить в виртульной дириектории веб-сервера.
